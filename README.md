# JS-Game-Dev-Hackathon
Phaser Starter Repo w/Express Server for my JS Game Dev Hackathon

## Getting Started

From within the root directory:
```sh
* npm install
```
Running the Server
```sh
* nodemon server.js
```
Controls in the demo are the up/left/right arrow keys
